css can be added to html Document in 3 ways

  1. Inline: Inline CSS contains the CSS property in the body section attached with the element known as inline
  2. Internal or Embedded: The CSS ruleset should be within the HTML file in the head section i.e the CSS is embedded within the HTML file.
  3. External: External CSS contains a separate CSS file that contains only style property with the help of tag attributes.
